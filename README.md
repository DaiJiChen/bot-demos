# QQ频道Go SDK使用示例

## demos目录说明

- ActiveMessage 主动消息推送
- HelloWorld 被动消息推送

## 运行说明
所有的demo都需要前置下面的操作：

- 申请[QQ机器人](https://bot.q.qq.com/#/home)，获取 `BotAppID`、`Bot Token`，修改`demos/config.example.yaml`为`demos/config.yaml`，并填入内容，格式参考：

```yaml
appid: 123456789
token: "O2AUl44m1mPu4jKVjAwpNtXXXXXXXXXX"
```
